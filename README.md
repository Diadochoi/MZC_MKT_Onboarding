# MegazoneCloud 마케팅 온보딩 플랫폼

한국어/영어 이중 언어 지원, 진행상황 자동 추적, 멘토 매칭, 피드백 시스템을 포함한 완전한 온보딩 플랫폼입니다.

## 파일 구조

```
vercel-onboarding/
├── index.html       # 온보딩 플랫폼 메인 파일
├── package.json     # Node.js 프로젝트 설정
├── vercel.json      # Vercel 배포 설정
└── README.md        # 이 파일
```

## 빠른 배포 (1분)

### 1. GitHub에 업로드
- GitHub 계정 생성 또는 로그인
- 새 저장소 생성: `megazone-onboarding`
- 위 파일 3개 업로드

### 2. Vercel 배포
- https://vercel.com 접속
- "Import Project" → GitHub 저장소 선택
- 자동 배포 시작

### 3. 배포 완료
- URL 확인: https://megazone-onboarding.vercel.app
- 신규입사자에게 공유!

## 기능

✅ 4주차 체계적 커리큘럼 (27개 미션)
✅ 진행상황 자동 추적 (로컬 스토리지)
✅ 실시간 진행률 그래프
✅ 한영 이중언어 지원
✅ 멘토 매칭 시스템
✅ 피드백 제출 기능
✅ 7개 팀 소개 및 정보
✅ 모바일 반응형 디자인
